A Journey into HTML and CSS

We started our journey exploring concepts of hierarchy and semantics, as well as how to insert and format content like images, videos, and audio, making the page more dynamic and engaging. Additionally, we worked with basic concepts of CSS, developing skills to set layouts and styles, and manipulate elements like margins, padding, colors, and fonts to give a personalized touch to pages. Essential Concepts in JavaScript

During my studies, I delved into several fundamental concepts of JavaScript, such as handling Strings and Numbers, Conditions, which enable the creation of logical flows based on different criteria, and practiced Control Flow and Loops, crucial for repeating actions and making decisions in code. I also examined the Browser Object Model (BOM) and the Document Object Model (DOM), which provide interaction with the browser and dynamic changes to content and style. Finally, I explored Functions, making the code more modular and manageable.
